1. Firebase-Ace

    Firebase as Database and Ace as Plaintext Editor. By default, Firebase runs inside an emulator, but feel free to update firebase.json to use it with actual database.

  2. Firebase-Monaco

    Firebase as Database and Monaco as Plaintext Editor. By default, Firebase runs inside an emulator, but feel free to update firebase.json to use it with actual database.

  3. Pusher-Monaco

    Pusher as Event Channel and Monaco as Plaintext Editor. By default, Firebase runs inside an emulator, but feel free to update firebase.json to use it with actual database.
