# `@otjs/plaintext`

> Exports Operational Transformation on Plain Text content.

## Usage

```ts
import { PlainTextOperation } from "@otjs/plaintext";

const operation = new PlainTextOperation();
```

**Note:** API Guidelines will be provided with the package.
